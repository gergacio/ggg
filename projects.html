The way converting type in js called COERCION.

  • + is overloaded operaor for concatenation.
  • string concatenation - implicit coercion
  • numeric coercion - Number(str)
  • boolean coercion - falsy/truthy concepts

Make sure types clear.

Checking for EQUALITTY with "==" and "==="

"==" allows cercion when types are different.

"===" disallows coercion.

IIFE: - wrap up function declaration and immediately invoke it with new block scope.
